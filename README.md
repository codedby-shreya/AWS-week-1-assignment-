## Task 2: Core AWS Services

### Amazon EC2 (Elastic Compute Cloud)

Amazon EC2 is a cloud computing service that provides scalable virtual servers for deploying and managing applications. Users can launch instances on demand and scale computing resources based on workload requirements.

### Amazon S3 (Simple Storage Service)

Amazon S3 is an object storage service used to securely store and retrieve data such as documents, images, videos, application files, and backups. It provides high durability, availability, and scalability.

### Amazon RDS (Relational Database Service)

Amazon RDS is a managed relational database service that simplifies database administration, including deployment, maintenance, backups, and scaling. It supports database engines such as MySQL, PostgreSQL, MariaDB, Oracle, and Microsoft SQL Server.

### AWS Lambda

AWS Lambda is a serverless computing service that allows developers to execute code without provisioning or managing servers. It automatically scales based on incoming requests and charges only for the compute time used.

### AWS Identity and Access Management (IAM)

AWS IAM enables secure access management for AWS resources by allowing administrators to create users, groups, roles, and policies with fine-grained permissions.

## Mini Project: IAM & Amazon S3 (Conceptual)

### Objective

- Create an IAM user with specific permissions.
- Create an Amazon S3 bucket.
- Upload static files.
- Configure public access.
- Test bucket policies.

### Conceptual Workflow

#### Creating an IAM User

1. Sign in to the AWS Management Console.
2. Navigate to **IAM**.
3. Select **Users**.
4. Click **Create User**.
5. Enter a username.
6. Assign the required permissions or policies.
7. Review the configuration.
8. Create the user.

#### Creating an Amazon S3 Bucket

1. Open the **Amazon S3** service.
2. Select **Create Bucket**.
3. Provide a globally unique bucket name.
4. Choose an AWS Region.
5. Configure bucket settings.
6. Create the bucket.
7. Upload files.
8. Configure bucket permissions if public access is required.
